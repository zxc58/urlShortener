# urlShortener
一個可以讓過長導致不好記憶或總是超出版面的網址轉換成短網址的app
![image](https://github.com/zxc58/urlShortener/blob/master/Pic.png)
## 環境/套件要求

node.js

mongodb

## 安裝/啟動
開啟終端機
git clone到本地端
```
$git clone https://github.com/zxc58/urlShortener.git
```
進入專案資料夾
```
$cd urlShortener
```
安裝必要套件
```
$npm install
```
use seed
```
npm run seed
```
node啟動
```
$npm start
```
